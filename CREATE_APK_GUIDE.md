# 🚀 СОЗДАНИЕ APK ФАЙЛА - ПОШАГОВАЯ ИНСТРУКЦИЯ

## Шаг 1: Установка EAS CLI

Откройте терминал на вашем компьютере и выполните:

```bash
npm install -g eas-cli
```

## Шаг 2: Создание аккаунта Expo (если нет)

Перейдите на https://expo.dev/ и создайте бесплатный аккаунт.

## Шаг 3: Войти в Expo

В терминале выполните:

```bash
eas login
```

Введите ваш email и пароль от Expo.

## Шаг 4: Подготовка проекта

1. Откройте терминал в папке `/app/frontend` вашего проекта

2. Обновите URL сервера в файле `.env`:
```bash
# Если у вас уже есть сервер
EXPO_PUBLIC_BACKEND_URL=https://ваш-домен.com

# Если пока тестируете локально
EXPO_PUBLIC_BACKEND_URL=https://private-messenger-136.preview.emergentagent.com
```

## Шаг 5: Инициализация EAS Build

```bash
cd /app/frontend
eas build:configure
```

При запросе выберите:
- `All` (для всех платформ) или `Android` (только Android)

Это создаст файл `eas.json` с конфигурацией.

## Шаг 6: Создание APK

Выполните команду для создания APK:

```bash
eas build --platform android --profile preview
```

**Что будет происходить:**

1. EAS спросит: "Generate a new Android Keystore?" - выберите **Yes**
2. EAS спросит: "Project name?" - введите **mychat** или любое имя
3. Начнется сборка на серверах Expo (10-20 минут)
4. Вы увидите ссылку на страницу сборки

**ВАЖНО:** Сохраните ссылку на страницу сборки!

## Шаг 7: Скачивание APK

Когда сборка завершится:

1. Откройте ссылку которую дал EAS
2. Или выполните: `eas build:list`
3. Нажмите кнопку **Download** рядом с вашей сборкой
4. Скачайте файл `.apk` на компьютер

## Шаг 8: Тестирование APK

### На своем телефоне:

1. Подключите телефон к компьютеру через USB
2. Скопируйте APK файл на телефон
3. На телефоне откройте файл APK
4. Разрешите установку из неизвестных источников (если попросит)
5. Установите приложение

### Через ADB (альтернатива):

```bash
adb install mychat.apk
```

## Шаг 9: Раздача друзьям

### Вариант 1: Прямая отправка
- Отправьте APK файл через Telegram/WhatsApp/Email
- Размер ~50-70 MB

1. Загрузите APK на ваш сервер:
```bash

```

2. Создайте простую HTML страницу для скачивания:

### Вариант 3: QR код

1. Создайте QR код со ссылкой на APK: https://www.qr-code-generator.com/
2. Покажите QR код друзьям
3. Они отсканируют и скачают

## Шаг 10: Обновление приложения

Когда нужно выпустить обновление:

1. Обновите версию в `app.json`:
```json
{
  "expo": {
    "version": "1.1.0"  // Было 1.0.0
  }
}
```

2. Пересоберите APK:
```bash
eas build --platform android --profile preview
```

3. Отправьте новый APK друзьям

## ⚠️ Важные замечания

### При установке APK:
- Android покажет: "Установка из неизвестных источников"
- Это НОРМАЛЬНО для приложений вне Google Play
- Нужно разрешить установку в настройках

### Первый запуск:
- Пользователи вводят имя и email
- Они сразу попадают в чаты
- Могут искать других пользователей

### Keystore:
- EAS автоматически создаст keystore
- Он хранится в облаке Expo
- Не теряйте доступ к аккаунту Expo!

## 🔥 Быстрые команды

```bash
# Проверить статус сборки
eas build:list

# Создать новую сборку
eas build --platform android --profile preview

# Посмотреть подробности конкретной сборки
eas build:view [BUILD_ID]

# Скачать APK напрямую (если знаете ID)
eas build:download [BUILD_ID]
```

## 🆘 Частые проблемы

### "Build failed"
- Проверьте `app.json` - нет ли ошибок
- Посмотрите логи сборки в веб-интерфейсе

### "Not logged in"
- Выполните: `eas login`

### APK не устанавливается
- Проверьте что разрешена установка из неизвестных источников
- Убедитесь что достаточно места на телефоне
- Попробуйте удалить старую версию

### Приложение крашится при запуске
- Проверьте что сервер запущен и доступен
- Убедитесь что `EXPO_PUBLIC_BACKEND_URL` правильный

## 📊 Мониторинг

После раздачи APK следите за:
- Количеством зарегистрированных пользователей
- Логами сервера (`/var/log/supervisor/backend.out.log`)
- Использованием MongoDB (`mongo --eval "db.stats()"`)
