# 📱 Быстрая инструкция: Создание APK файла

## 1. Установить EAS CLI

На вашем компьютере:

```bash
npm install -g eas-cli
```

## 2. Войти в Expo аккаунт

Если у вас нет аккаунта - создайте на https://expo.dev/

```bash
eas login
```

Введите email и пароль.

## 3. Обновить URL сервера

Перед созданием APK обновите файл `.env` в папке `frontend`:

```bash
# Замените на адрес вашего сервера
EXPO_PUBLIC_BACKEND_URL=https://ваш-сервер.com
```

## 4. Создать конфигурацию сборки

```bash
cd /app/frontend
eas build:configure
```

Выберите:
- Platform: `Android`
- Создаст файл `eas.json`

## 5. Создать production APK

```bash
eas build --platform android --profile production
```

Выберите при запросе:
- Build type: `APK` (не AAB)
- Generate a new keystore: `Yes`

Сборка займет 10-20 минут. Процесс идет на серверах Expo.

## 6. Скачать готовый APK

После завершения сборки:
- Откроется ссылка в браузере
- Или используйте команду: `eas build:list`
- Скачайте APK файл

## 7. Поделиться с друзьями

**Вариант 1: Прямая ссылка**
- Загрузите APK на свой сервер
- Настройте nginx для раздачи файла (см. DEPLOYMENT_GUIDE.md)
- Отправьте ссылку друзьям

**Вариант 2: Мессенджеры**
- Отправьте APK файл напрямую через Telegram/WhatsApp
- Размер ~50-70 MB

**Вариант 3: QR код**
- Создайте QR код со ссылкой на https://www.qr-code-generator.com/
- Распечатайте или покажите на экране

## 📝 Важно!

При установке APK пользователи увидят предупреждение Android:
- "Установка из неизвестных источников"
- Нужно разрешить установку в настройках

Это нормально для приложений вне Google Play Store.

## 🔄 Обновление приложения

Для выпуска новой версии:

1. Обновите версию в `app.json`:
```json
{
  "expo": {
    "version": "1.1.0"
  }
}
```

2. Пересоберите APK:
```bash
eas build --platform android --profile production
```

3. Отправьте новый APK друзьям

## 🆘 Проблемы?

**Ошибка: "Not logged in"**
```bash
eas login
```

**Ошибка: "Build failed"**
```bash
# Посмотреть логи
eas build:list
# Открыть детали сборки
```

**APK слишком большой**
- Это нормально для первого релиза (50-70MB)
- Expo включает все необходимые библиотеки
