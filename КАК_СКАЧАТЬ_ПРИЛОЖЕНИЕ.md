# 🚀 КАК СКАЧАТЬ ПРИЛОЖЕНИЕ - 2 СПОСОБА

## СПОСОБ 1: СОЗДАТЬ APK (для раздачи друзьям)

### На своем компьютере выполни:

```bash
# 1. Установи EAS CLI (один раз)
npm install -g eas-cli

# 2. Войди в Expo (или создай аккаунт на https://expo.dev/)
eas login

# 3. Перейди в папку проекта
cd /app/frontend

# 4. Запусти сборку APK
eas build --platform android --profile preview
```

### Что будет:
1. EAS спросит: "Generate new keystore?" → Выбери **Yes**
2. Начнется сборка на серверах Expo (~15 минут)
3. Откроется ссылка на страницу сборки
4. Когда готово - нажми **Download APK**

### Альтернатива - автоскрипт:
```bash
# Запусти готовый скрипт
cd /app
./build-apk.sh
```

## СПОСОБ 2: ТЕСТИРОВАТЬ БЕЗ APK (прямо сейчас!)

### На телефоне:

1. **Скачай Expo Go** из Google Play:
   https://play.google.com/store/apps/details?id=host.exp.exponent

2. **Открой QR код** который показывает сервер

3. **Отсканируй** через Expo Go

4. **Готово!** Приложение запустится

## 📥 ПОСЛЕ СОЗДАНИЯ APK

### Раздать друзьям:

**Вариант 1 - Прямая отправка:**
- Отправь APK файл через Telegram/WhatsApp
- Размер: ~50-70 MB

**Вариант 2 - Загрузка на сервер:**
```bash
# На своем сервере
scp mychat.apk root@твой-сервер:/var/www/html/

# Создай простую страницу download.html:
<a href="mychat.apk" download>Скачать Мой Чат</a>

# Отправь друзьям ссылку
```

**Вариант 3 - QR код:**
1. Загрузи APK на сервер
2. Создай QR код со ссылкой: https://www.qr-code-generator.com/
3. Покажи QR друзьям
